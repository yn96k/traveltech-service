## bookings
Хранит состояние распределенной транзакции (Saga). Отслеживает жизненный цикл бронирования во внешних системах.

| Имя атрибута | Тип данных | Constraint | Описание |
|--|--|--|--|
| id | UUID | PK, Not Null | Уникальный идентификатор бронирования |
| trip_id | UUID | Not Null | Идентификатор связанной поездки |
| user_id | UUID | Not Null | Идентификатор заказчика |
| saga_status | VARCHAR(50) | Not Null Check(status in ('PENDING', 'CONFIRMED','FAILED', 'COMPENSATING')) | Статус сессии |
| total_price | DECIMAL(12,2) | Not Null DEFAULT 0 | Итоговая стоимость |
| created_at | TIMESTAMPTZ | Not Null | Время инициации бронирования |

## booking_items
Детализация конкретных услуг внутри одного бронирования (билеты, номера).

| Имя атрибута | Тип данных | Constraint | Описание |
|--|--|--|--|
| id | UUID | PK, Not Null | Идентификатор позиции |
| booking_id | UUID | Not Null | Ссылка на родительское бронирование |
| provider_id | VARCHAR(100) | Not Null | Идентификатор внешнего сервиса для интеграционного шлюза |
| type | VARCHAR(50) | Not Null | Тип: FLIGHT, HOTEL, TRANSFER etc. |
| provider_status | VARCHAR(50) | Not Null | Технический статус внешнего провайдера |
